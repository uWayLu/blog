# Notion Blog 是什麼？

ijjk/notion-blog (opens new window)

A Next.js site using new SSG support with a Notion backed blog

將你的 Notion 做為文章管理的資料庫， notion-blog 透過 notion API (private, authed) 取得文章資料， 預渲染成靜態頁面，並部署到 Vercel (Optional)

# 按下 Deploy 後

  1. 記得填 NOTION_TOKEN 和 BLOG_INDEX_ID ，或者 deploy fail

  2. 手動建立文章列表吧， node scripts/create-table.js 無法正常工作

    notion-blog Issues (opens new window)
